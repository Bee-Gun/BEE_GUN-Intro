## 문제인식 (Problem)
**국내·외 시장(사회·경제·기술)의 문제점을 혁신적으로 해결하기 위한 방안 등을 기재**

의료, 금융, 제조업등 많은  국내산업전반적으로 인공지능이 접목되고있습니다.  현재, 많은 자본이 모이는 곳은 빠르게 AI기술이 도입되고 많은 인재들의 채용이 이뤄지고 있습니다. 하지만 농가, 장애우, 케어분야등 시장자본투입이 미비한곳은 상대적으로 발전이 더딘 경향이 있어 보입니다. 좀 더 주변의 소소하고 낮은곳에 눈을 돌려 인공지능이 접목되어야 한다고 생각합니다.

최근 중국에만 서식하는 **등검은말벌**의 유입으로 양봉농가가 어렵다합니다. 말벌 서식처도 사람이 거주하는 곳에 분포하며 높은 곳에 위치하며 양봉농가 뿐만아니라 여러 가지 문제를 일으킨다고합니다.

**양봉장가에 국한해서보면**, 말벌한마리가 벌통앞에서 꿀벌 수백여마리를 사냥합니다.  양봉업자는 다른 일을 하지 못하고 양봉장을 지키면서 말벌방어에 힘쓰는데, 그마저도 놓치게 되면 그해 수확량은 급감하게 됩니다.여기에 인공지능 인식기술과 약간의 임베딩이면 말벌을 인식하고 발멸까지 가능하리라는 생각으로 아이템을 개발하게 되었습니다.

## 기존대책
- 기존 말벌 퇴치기는 트랩을 설치, 쥐 끈끈이 등을 사용하여 말벌을 아날로그 적으로 퇴치
- 양봉농가에 인공지능 도입 자체가 새로운 시도이면서, 박멸시도하는 제품은 없음
- 안동대 이철희 교수의 말벌 분류 시스템이라는 논문이 있으나, 그사이 인공지능 모델들이 많이 개선되었고, 말벌인식에 국한되어있다.
- 말벌인식과 임베딩을 접목한 제품은 아직까지 없는것같다.
